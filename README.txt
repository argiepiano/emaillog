DESCRIPTION
===========
When maintaining an site where critical log entries needs to be notified to
site administrator, it would be great to have them by email.

This module will contain following features (in importance order):
  - #1816778: Send email when watchdog entry is entered
  - #1816786: Make alert severity level configurable
  - #1816790: Make email recipient configurable
  - #1816796: Make email template configurable
  
When all features above are implemented. Version 7.x-1.0-rc1 can be released.
